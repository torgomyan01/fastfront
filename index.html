#Get Start

The basis of FastFront's work is to speed up the work of FrontEnd programmers and have a faster result.

It has already been tested and the result is already visible. For example, if we work normally for 1 hour, it speeds up our work by 40%.

You just have to try it :)

Also, we have to write less CSS, which will contribute to the fast loading of the site, because it works more dynamically and there is no need to write CSS manually.

First let's check the marker property, there are 2 markers at the time of drawing,

! = !important, It will add !important at the end, eg !w-100 = width: 100px !important;

% = percentage, that is, the generated value will be equal to %, for example, %w-25 = width: 25%

#Flex

Here we should pay attention to the fact that we have reduced the values of Flex, which will help us to write shorter code, which is a great result.

  • sb => justify-content: space-between;
  • s -> align-items: start;
  • c -> align-items: center;
  • se -> justify-content: space-evenly;
  • sa -> justify-content: space-around;
  • fs -> align-items: flex-start;
  • fe -> align-items: flex-end;
  • jc -> justify-content: center;
  • jfs -> justify-content: flex-start;
  • jfe -> justify-content: flex-end;
  • as -> align-items: stretch;
  • ae -> align-items: end
